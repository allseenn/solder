## Сопротивление

$R=\rho\frac{l}{S}

- $R$ - электрическое сопротивление (Ом)
- $\rho$ - удельное сопротивление ($Ом\cdot м$)
- $l$ - длина проводника (м)
- $S$ - площадь сечения ($м^2$)

$l = \frac{RS}{\rho}$

$S = \frac{\rho l}{R}$

$\rho = \frac{RS}{l}$
